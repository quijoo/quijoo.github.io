个人开发机/服务器计划

在家里装一个服务器，是一个很酷的想法，他可以实现以下功能：

  1. AI 推理服务（自己可以做很多AI，然后提供服务）
  2. 家庭摄像头（监控家里的情况）
  3. 以Http/Socket/Rpc等形式提供各种小工具接口
  4. 托管博客
  5. ......

硬盘

由于一般的主机不提供容灾能力，所以数据使用 NAS 服务器：群晖NAS 预算 2000

GPU

由于需要 实现AI推理服务，所以服务器应该有一个 TPU/GPU/NPU，因为显卡这个东西，掉价很严重，在个人服务器上部署高端显卡是很亏的，更不要说 Nvidia A100， H100，Quadro 这样比较昂贵的显卡，个人 GPU 服务器不需要同时提供高并发的 AI推理请求，所以只需要用过气显卡即可：
例如：现在是 2022.10.24，现在GTX1066，1070ti,1080ti 的价格分别是 690，900，1600，并且个人服务器显卡不是刚需，可以等到显卡价格非常便宜的时候再更新换代，并且可以购买二手显卡。所以显卡预算基本就在 1000块左右了。
并且选择时要考虑 单位算力价格，单位算力功耗

CPU

查阅相关资料，服务器CPU一般采用精简指令集，使用较大的多级缓存，缓存命中较高，内存通道数更多（4/8），但是服务器CPU价格贵，并且需要专用的主板，好像不划算！
所以还是使用消费级CPU和主板，个人服务器上可能运行，带项目的编译，网站，AI推理容器，等任务。所以CPU应该购买 核心数比较多的，由于不运行游戏这样单核性能要求高的软件，所以不需要极致的单核性能，那AMD现在线程数量比较高的 AMD锐龙9 7950X处理器(r9)5nm 16核32线程4.5GHz 170W全新盒装CPU 但是这个好像功耗有点过高了，5500块

主板

主板就买接口丰富，可拓展性强的就行了，别的也不懂
那就 技嘉X670 吧 2500 块钱，但是好像现在没有支持 7950X + DDR5 的ITX 主板，等之后买的时候再看看吧！

内存

DDR5 就好了 64G 大概需要 5500块？？芝奇焰锋戟

硬盘

由于不装游戏，并且使用NAS服务，所以家里的二手硬盘 256G就够了

机箱

机箱主要得小，防尘，由于CPU散热打算使用水冷所以，散热没什么要求 SIXGE K3PRO 立式 itx 机箱 有点贵 1000块

水冷

[酷冷至尊 B280] 即可 300块

电源

750w 即可，预算 500

OS

Ubuntu 最熟悉。。但是可能 Windows 是最友好的。或许吧。。。用Windows 还是 Ubuntu 呢？
Linux 不能用 Visual Studio ，只有 Vscode，emm，好像也行，不过C++的话，就需要自己去配置 cmake这些，挺麻烦的。。。那就 Linux 吧，毕竟安装各种包什么的比较方便。而且可以在windows 上连接远程服务器开发？不过游戏开发好像不太能连接远程服务器。。。到时候再看吧/

总花费

2000 + 1000 + 5500 + 2500 + 5500 + 1000 + 300 + 500 = 18300
13000RMB大概，有点贵有一说一，而且还是约等于没有显卡！
可以暂时不要 NAS 服务器
CPU 可以买便宜一点的 ---> 7900X 4299
内存条可以买 32G 的暂时 ---> 2750

那就是

  1. 显卡 1000
  2. CPU 5500
  3. 主板 2500（买便宜一点的1500）
  4. 内存 2750
  5. 机箱 1000
  6. 水冷 + 电源 800
  7. OS 不要钱
    总共 12550 块，一个月攒 6000 就好！！！。
